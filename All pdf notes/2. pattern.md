## **Question: Pascal’s triangle (numbers)**

**Approach:**
We need to print Pascal’s triangle like this for n = 5:

        1
       1 1
      1 2 1
     1 3 3 1
    1 4 6 4 1

Outer loop runs from 0 to n-1 to handle rows.

Inner loop 1 prints spaces to center-align the triangle: n - i - 1 spaces.

Inner loop 2 prints binomial coefficients using nCr formula.

Use a variable num to store current value of the element in a row:

Start num = 1 for each row.

Update num = num \* (i - j) / (j + 1) for next element.

After each row, print a newline.

**Code:**

```cpp
#include <iostream>
using namespace std;

int main() {
    int n;
    cout << "Enter number of rows: ";
    cin >> n; // Input number of rows

    // Outer loop for each row
    for(int i = 0; i < n; i++) {
        // Print spaces to center-align
        for(int j = 0; j < n - i - 1; j++) {
            cout << " ";
        }

        int num = 1; // First element of each row is always 1
        // Inner loop to print elements in current row
        for(int j = 0; j <= i; j++) {
            cout << num << " "; // Print current number

            // Compute next number using nCr formula
            num = num * (i - j) / (j + 1);
        }
        cout << endl; // Move to next row
    }

    return 0;
}
```
